# Sistema de Venda de Ingressos 🎸

Projeto desenvolvido utilizando Bootstrap 5 durante um curso de curta duração voltado para desenvolvimento web responsivo.

O sistema simula uma página de venda de ingressos para eventos de rock, utilizando diversos componentes do Bootstrap para criação de interfaces modernas e responsivas.

## Tecnologias Utilizadas

- HTML5
- Bootstrap 5.3
- CSS Bootstrap Components

## Componentes Bootstrap Utilizados

- Accordion
- Badge
- Carousel
- Card Group
- Forms
- Input Group
- Buttons
- Cards

## Funcionalidades

- Cadastro de comprador
- Seleção de tipo de ingresso
- Exibição de informações do evento
- Carousel com imagens do evento
- Cards com tipos de ingressos
- Interface responsiva

## Objetivo do Projeto

O objetivo deste projeto foi praticar os principais componentes do Bootstrap e compreender a construção de layouts responsivos utilizando apenas classes utilitárias do framework.
